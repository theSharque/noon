# Полдень 21 века

Это проект старой игры, который я решил хоть как-то восстановить.
Для начала я верну всё как было, а дальше — если будет желание — буду развивать.

## Запуск

```bash
docker compose up -d --build
```

Сайт: http://127.0.0.1:8080/

## Структура

| Путь | Назначение |
|------|------------|
| `www/` | Игровой PHP/статика |
| `db/` | Миграции Flyway (схема + сиды) |
| `docker/` | Dockerfiles (php, nginx, cron, mysql) |
| `docker-compose.yml` | Стек: mysql, flyway, php, nginx, cron |
| `old_data/` | Архив исходников и старого хлама (не в git) |

## Готово

- Структура БД восстановлена по запросам и старым бэкапам (Flyway, UTF-8)
- Бэк перенесён в Docker, стабильный локальный запуск
- Перефигачена структура репозитория — в git только нужное (`www/`, `db/`, `docker/`)
- Выпилены платёжные шлюзы (Xsolla/Interkassa/WebMoney и т.п.); конфедераты — только ручным начислением админом
- Выпилены SAPE, аналитика и счётчики (Metrika, GA, Rambler, Openstat, LiveInternet, UpToLike)
- Выпилены CPA/партнёрки (luxup, 7offers, CityAds, WASD, inetrek)
- Почищен сайт: лишние меню/VK-вход, битые картинки галереи
- Удалён phpBB-форум (`www/info/`)
